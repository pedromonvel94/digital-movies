## Setup (Instalar dependencias)

Este backend está hecho con **Node.js** y **Express**.

### 1) Requisitos
- Tener **Node.js** instalado (incluye npm)

### 2) Instalar dependencias (incluye Express)
En la raíz del proyecto (donde está el `package.json`) corre:

```bash
npm install
```

### 3) Ejecutar el proyecto

Modo normal:

```bash
node index.js
```

Modo desarrollo (reinicia automático al guardar cambios):

```bash
node --watch index.js
```
